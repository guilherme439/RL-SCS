## Installation

It is recommended to create a virtual environment before installing the dependencies:

### 1. Create a virtual environment

```bash
python3 -m venv venv
```

### 2. Activate the virtual environment

On Linux/macOS:
```bash
source venv/bin/activate
```


### 3. Install dependencies

For regular use:
```bash
pip install -e .
```

For development:
```bash
pip install -e .[dev]
```
